
Hi. This is an ASCII mock up of a piece of 
software that helps people build software 
consensually.

When you see ~'~'~ and then a name and a colon, 
it means you're seeing what that person is 
seeing. This is what it looks like when Kynthia 
sees a button:

~'~'~ kynthia:

[ Button ]


* * * means we've moved on to the next screen.


This is a button someone is clicking on:

[ >Button< ]


This is something someone could type something 
into:

 Edit me!
-----------------------------------------


This means they tapped it:

 Edit| me!
-----------------------------------------


This is them typing stuff with a button to do 
something:

 Typing typing la la la
-----------------------------------------
 [ Do something ]


And lastly, a drop-down menu looks like this:

[ California v]
