Component Styles

This a minimal set of UI (user interface) controls for building applications that are about writing snippets of text and performing actions.

We start with a for the typeface: dark gray Helvetica: `style`

body, input, button, p {
  font-family: Helvetica;
  font-size: 18px;
  color: #555;
  -webkit-font-smoothing: antialiased;
}

Then in general we want to have a space around everything and mostly no borders on things: `style`

input, input[type=text], button, .thing, li, .multiple-choice, .row {
  border: 0px;
  padding: 9px 15px;
  display: block;
  margin-bottom: 15px;
}

input::-webkit-input-placeholder {
  color: #aaa;
}

Text inputs have just an underline: `style`

input[type=text] {
  border-bottom: 2px solid #bbb;
  width: 20em;
}

That's enough to to show a text input with a label: `client test`

dom.p('Software + Consent')
dom.input({type: 'text', placeholder: 'Write a story'})

Software+Consent

A really simple button: `style`

button, .multiple-choice {
  background: rgb(10, 209, 136);
  color: white;
  display: inline-block;
  margin-right: 15px;
  margin-bottom: 15px;
}

button:hover, .multiple-choice:hover {
  background-color: rgb(0, 226, 143);
}

Also applies to the multiple-choice boxes, which get a triangle added: `style`

.multiple-choice {
  padding-right: 45px;
  background-image: url('http://i.imgur.com/2NUM1zq.png');
  background-repeat: no-repeat;
  background-size: 24px 24px;
  background-position: right 9px center;
}

They work like this: `client test`

dom.button('Do not push')
dom.button({'class': 'multiple-choice'}, 'Choices')

We use hr elements to space things out:

hr {
  width: 100%;
  height: 15px;
  display: block;
  border: none;
  margin: 0;
}

And need a few other styles to make all of the paragraphs and stuff be spaced out well: `style`

p {
  margin: 15px 0;
}

body {
  margin: 15px;
  max-width: 26em;
}

p button {
  margin-bottom: 0;
}

form {
  margin: 0;
}

Lastly we have a couple classes to separate out independent chunks of content, a thing and a row: `style`

.thing {
  padding-bottom: 0px;
  box-shadow: 0px 2px 7px rgba(0,0,0,0.2);
}

.thing .thing, .row {
  background: #f9f9f9;
  box-shadow: none;
}

And we want buttons to be off to the right side when they're in a row: `style

.row .multiple-choice {
  float: right;
  margin-top: -9px;
  margin-right: 0;
}

Which let's us make a few nice boxes with stuff in them: `client test`

dom.p('Kynthia started a new project Software+Consent+Bacon. How would you like to split your time?')
[
  dom.div({'class': 'row'}, [
    dom.button({'class': 'multiple-choice'}, '50%'),
    'Software+Consent'
  ]),
  dom.hr(),
  dom.div({'class': 'thing'}, [
    dom.p('Cook some bacon'),
    dom.div({'class': 'thing'}, [
      dom.p(Erik ignored this'),
      dom.button('Seek consensus')
    ])
  ])
]
